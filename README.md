# configs
Fish & Bash aliases and functions for productivity improvement.<br>
Docker and Kubernetes autocompletion and aliases.

Make sure the files have proper PATH:
* `~/.aliases`
* `~/.bashrc`
* `~/.bash_aliases`
* `~/.config/fish/config.fish`
* `~/.config/fish/completions/`
* `~/.gitconfig`
